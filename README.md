# Vendor-Profitability-Insights
# 📊 Vendor Performance Analysis

## Overview

This project explores vendor performance data to understand sales trends, identify top-performing vendors, and highlight areas for improvement. It includes data analysis, visualization, and a dashboard for better decision-making.

## Objectives

* Analyze vendor-wise sales performance
* Identify high and low performing vendors
* Understand revenue distribution and trends
* Create a visual dashboard for insights

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Jupyter Notebook
* Power BI

## Project Structure

* `Exploratory Data Analysis.ipynb` → Data cleaning and initial exploration
* `Vendor Performance Analysis.ipynb` → Detailed analysis and insights
* `vendor_performance.pbix` → Power BI dashboard
* `vendor_sales_summary.csv` → Dataset
* Python scripts for data processing

## Key Insights

* A small group of vendors contribute a major share of total revenue
* Certain vendors show consistent growth over time
* Some vendors have high sales but lower efficiency

## Learnings

* Hands on experience with data cleaning and EDA
* Improved data visualization and storytelling skills
* Built an interactive dashboard using Power BI
